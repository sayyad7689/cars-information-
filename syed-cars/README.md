# 🚗 Syed Cars - Drive Your Dream

Premium online car marketplace built with React, Firebase, and Tailwind CSS with glassmorphism design.

## Setup

1. Clone repository
2. Run `npm install`
3. Create Firebase project at [console.firebase.google.com](https://console.firebase.google.com)
4. Enable **Authentication** → Email/Password
5. Create **Firestore Database** in test mode
6. Copy Firebase config to `.env`
7. Run `npm run dev`

## Firebase Setup

1. Go to Firebase Console → Create Project
2. Authentication → Sign-in method → Enable Email/Password
3. Firestore Database → Create Database → Start in test mode
4. Project Settings → Your apps → Web app → Copy config values to `.env`
5. Open app → Go to Admin Panel → Click "Seed Database" to populate cars
